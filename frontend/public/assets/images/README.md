# Images

This directory contains image files used throughout the application.

## Organization

- Logos
- Icons
- Background images
- User interface graphics
- Marketing materials

## Supported Formats

- PNG (for images with transparency)
- JPG/JPEG (for photos and graphics)
- SVG (for scalable graphics and icons)
- WebP (for optimized web images)

## Guidelines

- Optimize images for web (compress to reduce file size)
- Use appropriate image dimensions
- Consider using WebP format for better compression
- Name files descriptively (e.g., `logo-dark.png`, `hero-background.jpg`)
- Store different sizes in subdirectories if needed (e.g., `thumbnails/`, `full-size/`)
# Videos

This directory contains video files used throughout the application.

## Organization

- Hero section background videos
- Demo videos
- Tutorial videos
- Promotional content

## Supported Formats

- MP4 (recommended)
- WebM
- OGG

## Guidelines

- Keep video files optimized for web (compressed)
- Use appropriate resolutions (720p or 1080p recommended)
- Consider providing multiple formats for browser compatibility
- Name files descriptively (e.g., `hero-background.mp4`, `demo-tutorial.mp4`)
